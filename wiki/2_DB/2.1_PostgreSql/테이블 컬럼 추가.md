##### - 테이블 컬럼 추가 DML 쿼리
```SQL
 ALTER TABLE test_table
   ADD COLUMN if not exists test_colum_yn varchar(1) default 'N' null ;
```

- if not exists (컬럼명)
  -> 해당 테이블에 같은 이름으로 이미 존재 하고 있는 컬럼이 있으면 컬럼 추가가 실패한다.
  -> 컬럼 길이등 여러 설정은 관계없이 컬럼 명 중복 여부를 확인한다.
  ![Pasted image 20251204112753.png](../../99.%20File/Pasted%20image%2020251204112753.png)
- varchar(1) - 타입(길이)
  -> 해당 컬럼 데이터의 타입과 길이를 설정할수 있다.
  
- default 'N' - default (값)
  -> 해당 컬럼의 기본값을 설정 할수 있다.
  ![Pasted image 20251204130616.png](../../99.%20File/Pasted%20image%2020251204130616.png)
  -> 기존에 있던 데이터들은 default 값으로 정의 된다
  
- null - null 허용유무
  -> null 또는 not null 을 통해 해당 컬럼 데이터의 null 허용을 정의할수 있다.
