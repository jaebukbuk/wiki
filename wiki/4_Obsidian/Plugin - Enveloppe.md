### 📌 깃설정  
![Pasted image 20260126235239.png](../99.%20File/Pasted%20image%2020260126235239.png)  
1️⃣ 깃 파일은 wiki 하위 실제 파일만 적용한다. (.Obsidian 파일은 개인화 파일이 너무 많다.)  
2️⃣ 레파지토리를 불러오고 해당 레파지토리에 .obsidian 을 붙여놓고 git ignore 을 통해 변경점 안잡히게 설정한다.  
  
  
### 📌 플러그인 설정 - GibHub 설정  
![Pasted image 20260126233734.png](../99.%20File/Pasted%20image%2020260126233734.png)  
1️⃣ 저장소, GitHub 사용자 이름, 깃허브 토큰 등을 설정한다.  
2️⃣ 자동으로 풀리퀘스트 병합을 통해 바로바로 마스터 브렌치에 머지될수 있도록 한다.  
3️⃣ 테스트 연결을 통해 연결을 확인한다.

### 📌내용설정  
![Pasted image 20260126234439.png](../99.%20File/Pasted%20image%2020260126234439.png)  
1️⃣ 마크다운 강제 줄바꿈을 통해 해당 노트 줄바꿈이 구현되도록 설정한다. (마크다운은 줄바꿈이 없어서 이중 공백으로 처리해야한다.)  
2️⃣ 데이터뷰 무조건 온 -> 데이터뷰 스크립트가 아닌 출력결과를 깃에 연동한다. ( 해당 플러그인을 쓰는 이유 )  
3️⃣ 상대경로는 확인 필요 - 요거 켜야 폴더 트리 만들어지나..?
