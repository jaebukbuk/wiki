### 📌 개요  
1️⃣ JSON (JavaScript Object Notation) 은 ECMAScript에서 파생된,  
특정 프로그래밍 언어에 종속되지 않는 경량 텍스트 기반의 데이터 표현방식이다.  
👉 정확하게는 자바스크립트의 객체 타입(Object literals)에서 유래 하였으며   
구조화된 데이터를 직렬화 하는 텍스트 기반의 표현 방식 이다.  
   
2️⃣ JSON은 간단한 문법이 있으며, 이를 통해 구조화된 데이터를 유연하게 표현할수 있다.  
  
### 📌 JSON Data Type  
JSON 은 4개의 원시타입과 2가지 구조화 타입을 가진다.   
1️⃣ 원시타입 : 문자열 (Strings), 실수형(Numbers), 불리언(Booleans), 널(null)  
2️⃣ 구조화타입 : 객체(Objects), 배열(Arrays)
